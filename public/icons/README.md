|------|------|-------|

| icon-96x96.png | 96x96 | Android Chrome |
| icon-128x128.png | 128x128 | Android Chrome |
| icon-144x144.png | 144x144 | Android Chrome |
| icon-152x152.png | 152x152 | iOS Safari |
| icon-192x192.png | 192x192 | Android Chrome, PWA |
| icon-384x384.png | 384x384 | Android Chrome |
| icon-512x512.png | 512x512 | PWA Splash Screen |

## Apple Touch Icon
Also place `apple-touch-icon.png` (180x180) in the `/public` directory.

## Design Guidelines
- Use the BF Suma green (#228B22) as the primary color
- Include the hexagon (⬡) logo mark
- Ensure icons have proper padding for maskable icon support
- Test with the [Maskable Icon Editor](https://maskable.app/editor)

## Generation
You can generate all sizes from a single 512x512 source using:
- [PWA Asset Generator](https://github.com/nicedoc/pwa-asset-generator)
- [RealFaviconGenerator](https://realfavicongenerator.net/)
